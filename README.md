# AI Social Trend Intelligence Platform

An AI-powered social media analytics platform that analyzes YouTube trends, performs sentiment analysis, generates word clouds, and predicts future engagement metrics.

## Features
- Trending Analytics Dashboard
- Sentiment Analysis
- Word Cloud Generation
- Future Trend Prediction
- Viral Score Analysis
- PostgreSQL Database
- Streamlit Dashboard
- YouTube Data API Integration

## Tech Stack
- Python
- PostgreSQL
- Streamlit
- Pandas
- Plotly
- Scikit-Learn
- NLTK
- YouTube Data API
